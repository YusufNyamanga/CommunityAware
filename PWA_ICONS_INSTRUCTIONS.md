# Simple PWA Icon Placeholder

To complete the PWA setup, create these PNG files:

## For logo192.png (192x192 pixels):
Create a simple 192x192 orange square with white text "UA" 

## For logo512.png (512x512 pixels):
Create a simple 512x512 orange square with white text "UA"

## Quick Creation Method:
1. Use any image editor (Paint, GIMP, Photoshop)
2. Create a square canvas (192x192 and 512x512)
3. Fill with orange color (#FF8C00)
4. Add white text "UA" in the center
5. Save as PNG files in the public/ directory

## Alternative: Use Online Tools
- Canva.com
- Fotor.com
- Pixlr.com
